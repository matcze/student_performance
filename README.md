## Exam score predictions
 
This project aims to predict students’ exam scores based on various input features using machine learning techniques. The problem is modeled as a regression task, where the target variable is a numerical value ranging from 0 to 100.


## 📁 Dataset

The dataset is sourced from the [Kaggle Student Exam Performance](https://www.kaggle.com/datasets/grandmaster07/student-exam-performance-dataset-analysis).

This dataset contains only a single dataset and is not pre-divided into training and testing subsets, unlike many Kaggle datasets.

### Main dataset

* 6607 observations
* 19 input features
* 1 target variable (`Exam_Score`)
* Total: 20 columns
